#  Phishing Attack Simulation & Detection

## Objective
Simulate a phishing attack in a controlled lab environment to:
- Understand phishing techniques
- Create phishing email template
- Setup fake landing page
- Send test phishing emails
- Track responses
- Identify phishing indicators
- Learn prevention techniques
- Document findings

## Tools Used
- GoPhish
- Kali Linux
- Test email accounts (lab environment only)

## Workflow
1. Create phishing email template
2. Setup landing page
3. Launch test campaign
4. Track victim interaction
5. Analyze phishing indicators
6. Prepare report
